A free, cross-platform tool (Windows, macOS & Linux) that scans your PC like an OBD2 scanner for cars — it scores 13 areas of your computer's health and tells you the best upgrade to make each one faster.

What it checks

🧠 Memory (RAM)

Total RAM vs. how you use the PC, plus live memory and commit pressure.

💾 Storage Space

Free space on your system drive and how much junk is clogging it.

⚡ Disk Speed & Health

HDD vs SSD on the boot drive, SMART health, and TRIM.

🔥 CPU

Cores, live load, clock behavior, and a motherboard-compatible upgrade pick.

🎮 Graphics (GPU)

Dedicated vs integrated, VRAM, and graphics-driver age.

🚀 Startup & Boot

How many programs launch at boot and drag down your login time.

⚙️ Background Apps

Third-party services and scheduled tasks running in the background.

🌡️ Power & Thermals

Power plan, battery state, and heat/throttling when Windows exposes it.

🔧 Drivers & Devices

Device Manager errors, old key drivers, and BIOS age.

🔄 Windows Updates

Pending restarts and how far behind Windows Update is.

🌐 Network & Browser

Link speed, Wi-Fi signal, browser extensions and cache size.

📈 Stability Logs

Recent disk, hardware (WHEA), blue-screen, and app-crash events.

🛡️ Security & Malware

Antivirus status plus a scan for suspicious files in risky locations.

How it works

  1. Scan. Run ShmagStick — it reads your PC's vitals in a few seconds. It only diagnoses; it never changes anything.
  2. Score. Each area gets a 0–100 grade, tuned to your device type (desktop or laptop) and whether you use the PC for everyday tasks, gaming, or a workstation.
  3. Upgrade. For each area scoring below 85, it suggests the best part for your hardware — and never recommends a desktop graphics card or a CPU swap on a laptop — then links you straight to it.

Recommended upgrades

When a section scores below 85 and a part can fix it — more RAM, an SSD, a graphics card — ShmagStick points you to the best-rated option that fits your machine.
